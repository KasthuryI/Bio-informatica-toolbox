TRIMMING OPTIONS



CROP: Cuts the read to a specified length.




MINLEN: Drop the read if it is below the specified length.




SLIDINGWINDOW: cuts if the average quality within the window is bellow a threshold (SLIDINGWINDOW:4:15) The first argument is the size of the window. The argument is the threshold quality.
